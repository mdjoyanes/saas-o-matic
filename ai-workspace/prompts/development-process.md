# AI Development Process

## Development Approach

The project was developed using AI as a coding assistant while maintaining control over architecture, business logic and code quality.

The workflow followed these steps:

1. Define requirements and business rules.
2. Design database structure.
3. Define API contracts.
4. Implement backend services.
5. Implement frontend components.
6. Test and improve features.


## Prompt Examples

### Backend Architecture

Prompt:

"Design a modular Flask REST API architecture for a SaaS billing simulator using SQLite. Separate models, routes and services following clean code principles."


### Pricing Algorithm

Prompt:

"Implement a tiered pricing algorithm where users are charged progressively depending on different ranges. The solution must be readable and easy to test."


### Frontend Dashboard

Prompt:

"Create a responsive React dashboard for managing SaaS customers. Include reusable components, API integration and clear state management."


### Code Review

Prompt:

"Review this implementation and identify possible bugs, security problems and improvements following professional software engineering practices."
