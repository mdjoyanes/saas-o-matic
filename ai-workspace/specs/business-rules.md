# Business Rules - SaaS-O-Matic


## Billing System

The application calculates monthly SaaS subscription costs using tiered pricing.


## Pricing Tiers

Tier 1:
0-10 users:
10€ per user


Tier 2:
11-50 users:
8€ per additional user


Tier 3:
More than 50 users:
5€ per additional user



Example:

15 users:

10 users x 10€ = 100€

5 users x 8€ = 40€

Total base price = 140€
